Ajuda & Solução de Problemas

Dicas para Firefox e funcionalidades do site

🔧 Firefox — Liberar Mixed Content (HTTPS → HTTP)

O Firefox bloqueia chamadas HTTP quando a página está em HTTPS. Siga os passos:

1
Na barra de endereço, digite about:config e pressione Enter.
2
Clique em "Aceitar o risco e continuar".
3
Na caixa de pesquisa, cole: security.mixed_content.block_active_content
4
Clique no ícone ⇄ à direita para mudar o valor de true para false.
5
Recarregue esta página. O KOReader agora pode ser acessado.
⚠ Atenção: Esta configuração vale para todas as páginas no Firefox. Reverta para true quando não estiver usando o KOReader Remote.
📋 Clipboard — Limitação no Firefox

O botão "Ler Clipboard" usa a Clipboard API. No Firefox via HTTPS, esta API exige permissão explícita do usuário.

1
Clique no botão "Ler Clipboard" — o Firefox vai exibir um popup de permissão na barra de endereço.
2
Clique em "Permitir" no popup. A permissão fica salva para este site.
3
Se não aparecer popup: vá em about:preferences#privacy → Permissões → Clipboard → adicione este site manualmente.
💡 Dica: Abrindo via file:// (duplo clique no arquivo), o clipboard funciona automaticamente sem pedir permissão — tanto no Firefox quanto no Chrome.
⚡ Solução mais simples para tudo

Abra o index.html com duplo clique no explorador de arquivos. A barra de endereço mostrará file:///C:/.... Nesse modo, Mixed Content e permissão de clipboard não existem — funciona 100% em qualquer browser.
