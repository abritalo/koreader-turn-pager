Setup & Endpoint

1
Ative o KOReader HTTP Inspector e ligue-o (a porta padrão é 8080, mas confirme lá). Menu (4º ícone, pessoa+engrenagem) → Mais ferramentas → KOReader HTTP Inspector

Dica: ative "Auto start HTTP server" para iniciar automaticamente.

2
Pegue o IP do Kindle: abra Informações de rede e copie o valor de IP. Menu (3º ícone, engrenagem) → Rede → Informações de rede
3
Digite o IP:Porta abaixo — armazenamos só neste browser.
4
Se seu celular/tablet travar a tela rápido, aumente o tempo de bloqueio para o controle continuar visível.
